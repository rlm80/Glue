Installation and usage

  1. Download the latest version from Github.
  2. Unpack on your web server, preferably outside the document root.
  3. Include the file bootstrap.php located at the root of the glue directory. This will register the class auto-loader so you'll never have to include anything else.
  4. At the beginning of every file (PHP import rules work per file) where you want to use the database library, add the import rule use Glue\DB; . If you wish to use the ORM, add use Glue\ORM; .
  5. Call the static methods DB::... and ORM::... to use the library.
