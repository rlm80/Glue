Select

Aggregates the following builders : boolean expressions (where, having), join expression (from), order by, group by, select list.

Methods

All methods return $this unless specified otherwise.

Methods that target the from clause :

  • ->from() : without parameters, returns the from clause; otherwise forwards call to ->init() function of the from clause.
  • ->inner(), ->right(), ->left(), ->comma() : forwards call to from clause.
  • ->on() : without parameters, returns the last ON clause; otherwise forwards call to ->and() function of the from clause.
  • ->andon(), ->oron() : forward call respectively to ->and() and ->or() function of the from clause.

Methods that target the where clause :

  • ->where() : without parameters, returns the where clause; otherwise forwards call to ->and() function of the where clause.
  • ->andwhere(), ->orwhere() : forward call respectively to ->and() and ->or() function of the where clause.

Methods that target the having clause :

  • ->having() : without parameters, returns the having clause; otherwise forwards call to ->and() function of the having clause.
  • ->andhaving(), ->orhaving() : forward call respectively to ->and() and ->or() function of the having clause.

Other methods that forward to aggregated builders :

  • ->orderby() : without parameters, returns the order by clause; otherwise forwards call to ->orderby() function of the order by clause.
  • ->groupby() : without parameters, returns the group by clause; otherwise forwards call to ->groupby() function of the group by clause.
  • ->columns() : without parameters, returns the select list; otherwise forwards call to ->columns() function of the select list.

Other methods :

  • ->limit() : limit getter/setter.
  • ->offset() : offset getter/setter.
  • ->unique() : uniqueness of rows getter/setter.
  • ->distinct() : same as calling ->unique(true).
