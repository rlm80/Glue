Insert lists

Factory : DB::insertlist()

Represents a comma separated list of column names, as required in an insert query : "id", "login", ...

Item class

Item has a single property (the column name) and a single method to set/get it : ->column()

Builder class

Builder : Fragment_Builder_InsertList

The builder has only one method (returns $this) : ->columns(). It adds a list of columns to the insert list. It can take any number of string parameters, or an array of strings.

Examples

{% highlight php %} columns('column3', 'column4'); // Output SQL : echo db::cn()->compile( $f ); // Prints "column1", "column2", "column3", "column4" ?> {% endhighlight %}
