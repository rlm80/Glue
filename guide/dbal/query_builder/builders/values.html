Values

Factory method : DB::values()

Represents a comma separated list of rows of values, as used in insert queries : (1,'test'),(2,'qsdf'), ...

Item class

Item : Fragment_Item_Values

The item has one property (an array of values or fragments) and one method : ->values() which is the array of values getter/setter.

Builder class

Builder : Fragment_Builder_Values

It has only one method : ->values(). It accepts either :

  • a list of any number of values/fragments (1 call defines 1 row of values),
  • an array of values/fragments (1 call defines 1 row of values),
  • an array of arrays of values/fragments (1 call defines many rows of values).

Examples

{% highlight php %} values(array(2, 'test2')) ->values( array( array(3, 'test3'), array(4, 'test4'), ) ); // Output SQL : echo db::cn()->compile( $f ); // (1,'test1'),(2,'test2'),(3,'test3'),(4,'test4') ?> {% endhighlight %}
