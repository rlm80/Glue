Fragments

Queries are represented as trees of fragments.

Fragments are objects that represent specific pieces of an SQL query in a database agnostic way (i.e. in a way independent of the RDBMS in use). Boolean expressions, boolean operands, order by clauses, join expressions and even queries, all of these are represented as fragments. Fragment classes may be found in the classes/system/db/fragment/ folder.

Compiling fragments

Fragments can be compiled into SQL strings according to the SQL dialect of some connection by passing them to the ->compile() function of that connection :

{% highlight php %} where("$u->login = ?", 'Mike')->columns($u->login); // Compile data structure against default connection to view generated SQL : echo db::cn()->compile($query); ?> {% endhighlight %}
