The following explanation is tailored for the DB library, but it applies to the ORM as well. Just replace "DB" by "ORM".

Defining options

All available options are protected properties of the \Glue\DB\Config class. They are assigned default values on declaration. If these default values don't suit your needs, you must override the \Glue\System\DB\Config class with your own copy by creating a \Glue\User\DB\Config class. A file that does just that is included with the library so you can start from there :

{% highlight php %} some_option = 'some ' . 'value'; $this->another_option = 1 + 1; } } ?> {% endhighlight %}
