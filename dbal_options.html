Defining options

There is no config file. All available options are protected properties of the \Glue\DB\Config class. They are assigned default values on declaration. If these default values don't suit your needs, you must override the \Glue\System\DB\Config class with your own copy by creating a \Glue\User\DB\Config class. Actually a sample file classes/user/db/config.php is included with the library so you can start from there.
