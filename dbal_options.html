Defining options

There is no config file.

All available options are protected properties of the main database class ~\Glue\System\DB\DB . They are assigned default values on declaration. The best way to change these values is to override the ~\Glue\System\DB\DB system class with your own copy by creating a ~\Glue\User\DB\DB class like so :

{% highlight php %} some_other_option = 'some ' . 'other ' . 'value'; $this->yet_another_option = 1 + 1; } } ?> {% endhighlight %}
