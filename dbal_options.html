Defining options

There is no config file.

All available options are protected properties of the main DBAL class \Glue\System\DB\DB . They are assigned default values on declaration. If these default values don't suit your needs, the best way to change them is to override the \Glue\System\DB\DB class with your own copy by creating a \Glue\User\DB\DB class.
