What it is

This is a combination of two projects :

  • a database abstraction library that extends PDO,
  • an object relational mapping library based on the data mapper pattern (in the works, not available yet).

Database abstraction library

For the most part this is a query builder that makes it possible to compose queries as a sequence of PHP function calls, rather than manipulating SQL code as strings :

{% highlight php %} columns($u->password, $u->email) ->where("$u->login = ?", 'Mike'); ?> {% endhighlight %}

There are two benefits in writing SQL queries this way :

  • queries are portable accross various database backends,
  • queries can be composed bits by bits, in no particular order.

Object relational mapping library (not available yet)

The ORM is built on top of the DBAL. It makes it possible to manipulate model objects only, never having to worry about the way they are stored into, and retrieved from, the database.
