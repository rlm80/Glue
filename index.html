Glue is a combination of two projects : a database abstraction library and an object relational mapping (ORM) library.

Database abstraction library

For the most part this is an expressive SQL query builder that makes it possible to compose queries as a sequence of PHP function calls, rather than manipulating SQL code as strings :

{% highlight php %} columns($u->login, $u->password) ->where("$u->login like ?", 'Abc%') ->orderby($u->login) ->execute(); ?> {% endhighlight %}

There are two benefits in writing SQL queries this way :

  • queries are portable accross various database backends,
  • queries can be composed bits by bits, in no particular order, making it easy for more than one object to contribute something to them.

ORM

The ORM is built on top of the DBAL. It makes it possible to deal with model objects only, never having to worry about the way they are stored into, and retrieved from, the database.
