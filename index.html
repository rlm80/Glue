Glue is a combination of two projects : an SQL query builder and an object relational mapping library.

The query builder makes it possible to compose queries with PHP function calls, rather than actually writing SQL code :

{% highlight php %} columns($u->login, $u->password) ->where("$u->login like ?", 'Abc%') ->orderby($u->login) ->execute(); ?> {% endhighlight %}

There are two benefits in writing SQL queries this way :

  • queries are portable accross various database backends,
  • queries can be composed bits by bits, in no particular order, making it easy for more than one object to contribute to them.

The ORM is built on top of the query builder. It makes it possible to deal with model objects only, never having to worry about the way they are stored into, and retrieved from, the database.
